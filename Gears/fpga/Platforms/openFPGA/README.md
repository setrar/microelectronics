# openFPGA

## Run

```sh
nextpnr-ice40 --version
```
> "nextpnr-ice40" -- Next Generation Place and Route (Version 20241022 (9c2d96f8))

## Install

- [ ]  Tap

```sh
brew tap twam/openfpga
```

- [ ] nextnpr

```sh
brew install nextpnr --with-gui --without-arch-ecp5
```

- [ ] icestorm (icepack)

```sh
brew info icestorm
```
